# iflyjs

做更好用的 JS 函数库, 值得前端开发者信任的 JS 函数库！
详细文档，请移步观看：https://ifly.yunfei.show/

## iflyjs 安装

```js
npm i iflyjs
```

## iflyjs 使用

```js
import { getValueType } from 'iflyjs'
getValueType(undefined)
```

## 技术栈

TypeScript + Vite
